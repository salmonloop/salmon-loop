## 2025-04-25 - Structural Focus Indicators in Selection Lists

**Learning:** When creating terminal selection lists using Ink, relying solely on color (like green text) to indicate the active selection can be problematic for colorblind users or when terminal color support is limited.

**Action:** Always include a structural or character-based focus indicator (e.g., `❯ `) alongside color changes in list components (like `CommandInput`'s selection mode) to ensure the interface remains accessible and intuitive for all users.
