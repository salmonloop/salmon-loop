## 2024-05-10 - Standardizing Structural Focus Indicators
**Learning:** We observed that different list selection components in the terminal UI used different selection indicators (e.g., `│ ` vs `❯ `) and varying color feedback. Relying solely on color changes or inconsistent characters like `│ ` (which looks like a border) can confuse users, particularly for colorblind accessibility or quick visual scanning. The `❯ ` character provides a strong structural, directional affordance that clearly indicates the selected item.
**Action:** When building command line selection interfaces using Ink, always use the explicit structural focus indicator `❯ ` in `COLORS.semantic.cyan` (along with bolding if applicable) to ensure a consistent, accessible focus indication that doesn't rely solely on color changes.
