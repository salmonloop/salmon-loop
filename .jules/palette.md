## 2024-05-18 - Consistent Character-Based Focus Indicators
**Learning:** Relying solely on color or subtle structural changes (like `│`) for list selection can be unclear and inaccessible, especially for colorblind users navigating CLI interfaces. The `❯` character provides an explicit, consistent directional cue that clearly indicates focus.
**Action:** When designing CLI selection lists, always pair color changes with explicit structural or character-based indicators (specifically `❯`) to signify focus uniformly across the application's components.
