## 2024-05-28 - Explicit Keyboard Hint Format
**Learning:** For terminal CLI UI interfaces built with Ink, verbose sentence-based navigation hints ("Use Up/Down to navigate...") consume excessive horizontal space and reduce scanability.
**Action:** Consistently use explicit, symbol-based microcopy (e.g., `↑↓ nav · space toggle · ↵ confirm · esc cancel`) across command input overlays and selection lists.
