## 2024-04-28 - Structural Focus Indicators in Terminal UIs
**Learning:** For list selections in Ink terminal UI components, relying solely on color changes (like turning text green) to indicate focus or selection is insufficient for accessibility, particularly for colorblind users. Structural or character-based focus indicators must be included alongside color changes.
**Action:** Always include an explicit character prefix (e.g., `❯`) to indicate the currently focused item in command-line selection interfaces.
